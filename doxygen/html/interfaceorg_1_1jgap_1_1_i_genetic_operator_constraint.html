Descripción detallada

Interface for a constraint checker that determines whether a genetic operation should be executed for a given list of chromosomes. it is possible considering the caller. With that one could state that the two chromosome with classes ChromosomeClass1 and ChromosomeClass2 should not be considered for CrossoverOperator but for MutationOperator.
