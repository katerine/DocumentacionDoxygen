Descripción detallada

This class provides an implementation of the classic "Make change" problem using a genetic algorithm. It is equivalent to the example in class MinimizingMakeChange in the same package, except that it uses a MapGene.

A MapGene allows to specify discrete values for possible solutions. Instead, an IntegerGene (used in the original example) uses a range of values.
