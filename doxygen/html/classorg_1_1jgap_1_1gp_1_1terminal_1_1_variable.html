Creates an instance of a Variable. If a Variable of that name already exists, it is returned. Otherwise a new instance is created, its value is initialized to null, and it is placed into the static hashtable for later retrieval by name via getVariable.

Parámetros
a_confthe configuration to use
a_namethe name of the Variable to create
a_typethe type of the Variable to create

the variable object created
